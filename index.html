Terraform

Terraform is an open-source infrastructure as code software tool that enables provisioning and adapting virtual infrastructure across all major cloud providers.

GitHub & Git

GitHub is a platform for version control and collaboration, allowing multiple people to work on projects at once. Git is a distributed version control system to track changes in source code during software development.

Azure DevOps CI/CD

Azure DevOps provides a set of development tools for software development, including CI/CD pipelines, version control, and project management.

Docker & Kubernetes

Docker is a platform for developing, shipping, and running applications inside containers. Kubernetes is an open-source system for automating the deployment, scaling, and management of containerized applications.

Ansible

Ansible is an open-source software provisioning, configuration management, and application-deployment tool enabling infrastructure as code.

Monitoring Tools

Tools like Azure Monitor, Prometheus, Grafana, and Promtail are used for monitoring and logging, providing insights into application performance and health.

Security Tools

Tools such as Checkov, TFSec, Terrascan, and TFLint are used for static code analysis to identify security vulnerabilities in infrastructure as code.

Helm & ArgoCD

Helm is a package manager for Kubernetes, and ArgoCD is a declarative, GitOps continuous delivery tool for Kubernetes.
